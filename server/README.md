# Run the app using following steps


 ```
git clone https://github.com/mukul7661/Overbooked/edit/main/README.md
cd Overbooked
docker-compose up --build

# Frontend will be available on port 5173
# Backend app will be available on port 8000
# LLM service will be running on port 8080

# Demo - https://www.loom.com/share/1316606a8f204d8db8a757de98564808

# To run tests
npx jest src/users.test.ts
npx jest src/recommendations.test.ts
```
